# Investment-Preference-Analysis-using-Excel
This project focuses on understanding investment choices and preference reasons among male and female investors using Microsoft Excel. The goal of the project is to practice basic data analysis skills such as grouping, comparison, and visualization using Pivot Tables.

**Objective**
To study different types of investment choices
To compare investment preferences between male and female investors
To understand common reasons behind investment decisions

**Description**
Worked with a structured investment dataset to explore investor preferences.
Used Pivot Tables to group and compare:
Investment types
Preference reasons
Gender-based differences
Created simple charts to visualize:
Distribution of investment choices (Pie Chart)
Comparison between categories (Bar Chart)
Basic time-based trends (Line Chart)
Reviewed the results to identify clear patterns and differences in investment behaviour.
